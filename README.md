# Simple-Event-App-Nodejs

A simple Event Management web application built with **Node.js**, **Express**, and **EJS templating engine** that supports full **CRUD operations** (Create, Read, Update, Delete). The app also features **authentication**, **form validation**, and a **responsive UI** using Bootstrap.

---

## 🛠️ Technologies Used

- **Node.js** – Runtime environment for building server-side applications.
- **Express.js** – Web framework for Node.js.
- **EJS** – Embedded JavaScript templating engine.
- **Passport.js** – Authentication middleware for Node.js.
- **Bcrypt.js** – Library to hash and compare passwords securely.
- **Express-Validator** – Middleware for validating and sanitizing user input.
- **Bootstrap** – For responsive front-end design.

---

## 📦 Features

- 🔐 User authentication (login, registration, logout)
- 📝 Create new events
- 👁️ View event details
- ✏️ Edit existing events
- ❌ Delete events
- ✅ Server-side validation using express-validator
- 🔒 Password encryption with bcrypt
- 💬 Flash messages for success/error feedback
- 📱 Clean and responsive UI using Bootstrap
